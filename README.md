# ARCHIVED -- Moved to ssilvius/legion

This repository has been archived. The Legion plugin for Claude Code is now maintained in the main legion repository:

**https://github.com/ssilvius/legion**

## Installation

```
/plugin marketplace add ssilvius/legion
/plugin install legion
```

## Migration

If you have this marketplace installed, remove it and switch to the new source:

```
/plugin marketplace remove claude-legion-plugins
/plugin marketplace add ssilvius/legion
/plugin install legion
/reload-plugins
```
